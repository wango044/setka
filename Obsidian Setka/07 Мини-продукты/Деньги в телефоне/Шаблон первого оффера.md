Цель: дать человеку первый понятный оффер, который можно отправить потенциальному клиенту без спама и стыда.

## Обещание продукта

За 20 минут ты превратишь "я вроде что-то умею" в конкретное предложение: кому помогаешь, какую задачу делаешь и какой результат отдаешь.

## Формула

```text
Я помогаю [кому] сделать [задача],
чтобы получить [понятный результат]
за [срок/формат].
```

Пример:

```text
Я помогаю маленьким Telegram-каналам оформить 5 постов в единый стиль,
чтобы канал выглядел аккуратнее и понятнее для новых подписчиков
за 1 день.
```

## Шаг 5. Проверка оффера

Ответь честно:

| Вопрос | Да/нет |
|---|---|
| Понятно, кому я помогаю? | |
| Понятно, что я сделаю? | |
| Понятно, какой будет результат? | |
| Можно сделать это быстро? | |
| Я не обещаю доход и магию? | |

Если где-то "нет" — оффер мутный. Упрощай.
